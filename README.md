# 🗼 东京涩谷 TokyoShibuya

> 高度还原的东京涩谷城市地图 | 网易MC Studio基岩版 | 创造模式

## 📖 项目介绍

这是一个为网易MC Studio基岩版开发的沉浸式涩谷城市地图，以真实东京涩谷为蓝本，还原地标建筑、街道、商业设施等。601x601的大尺度地图，让玩家仿佛在真实的涩谷街头漫步。

## ✨ 功能特色

### 🏗️ 建筑系统（14个建筑）
- **涩谷109大厦** - 外观+内部5层完整还原
- **涩谷十字路口** - 斑马线+交通信号灯
- **八公像** - 忠犬八公铜像（金块+铁块）
- **涩谷站** - 地铁站站台+闸机+站牌+座椅
- **便利店** - 货架+收银台+自动售货机
- **咖啡厅** - 吧台+咖啡桌+咖啡杯
- **拉面店** - 厨房+座位+拉面碗
- **卡拉OK** - 舞台+音响+霓虹灯
- **涩谷酒店** - 5层大堂+客房（新增）
- **涩谷公寓** - 5层公寓单元+阳台（新增）
- **展望塔** - 56格高螺旋楼梯+观景台（新增）
- **樱花公园** - 樱花树+石子路+长椅（新增）
- **涩谷神社** - 鸟居+本殿+参道+灯笼（新增）

### 🛣️ 街道系统
- **涩谷通**（主干道）- 东西向7格宽
- **中心街** - 南北向5格宽商业街
- **道玄坂** - 西南延伸斜坡
- **井之头通** - 平行商业街
- **樱花坂** - 粉色混凝土樱花路
- 所有街道配备路灯

### 🤖 NPC系统
- 15个角色（向导、店员、咖啡师、厨师、站务员等）
- AI行为：按时段移动、对话系统
- 分布在各个建筑和街道

### 🎯 任务系统
- 5个任务（初来乍到/美食猎人/职场新人/收藏大师/城市达人）
- 进度追踪、奖励系统

### 💰 经济系统
- 商店、银行、就业（6种岗位）、市场
- ATM、工资、交易功能

### 🚃 交通系统
- 地铁（3条线路）、公交（3条线路）、出租车（7个目的地）

### 🌤️ 环境系统
- 时间系统：早晨/正午/傍晚/深夜四阶段
- 天气系统：晴/雨/雪/雷暴
- 活动系统：樱花祭/夏日祭/秋叶祭/灯光节/新年

### 🎮 收集系统
- 12个印章（对应12个地标建筑）

## 🎮 使用方法

### 推荐生成顺序
1. `shibuya_core` - 生成地面
2. `shibuya_landmarks` - 生成十字路口+八公像
3. `shibuya_buildings` + `shibuya_109_interior` - 生成109大厦
4. 依次生成：便利店、地铁站、咖啡厅、拉面店、卡拉OK
5. 生成新建筑：酒店、公寓、展望塔、樱花公园、神社
6. `shibuya_npc_system` - 生成NPC
7. `shibuya_collectibles` - 生成印章
8. 启用系统：经济、交通、时间、天气、活动

### 游戏内命令
- `function shibuya:menu` - 打开主菜单
- `function shibuya:world/generate_all` - 生成世界
- `function shibuya:buildings/generate_all` - 生成所有建筑
